Java-Based Chat Simulator:

The Simple Chat / Message Simulator is a Java-based desktop application that simulates one-to-one chat conversations in a graphical environment. It allows users to send messages to predefined contacts and receive automated replies in real time. The system is designed using multiple software design patterns to ensure clean architecture, modularity, and scalability. A single central chat engine manages all message routing using the Singleton pattern, while the Observer pattern keeps the user interface updated instantly when new messages arrive. Message creation is handled through a Factory pattern, chat sessions are constructed using the Builder pattern, and message appearance is enhanced using the Decorator pattern without modifying the core message logic. Overall, the application demonstrates how object-oriented design patterns can be effectively applied to build a maintainable, extensible, and user-friendly chat system.

Lanuages & Technologies Used:

  • Java (JDK 17+)
  • Java Swing (Framework)
  • Singleton Design Patters
  • Observer Design Patters
  • Factory Design Patters
  • Builder Design Patters
  • Decorator Design Patters
