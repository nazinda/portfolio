Priority-Based To-Do list:

This project is a console-based task management system developed in Java that helps users organize tasks based on priority levels: High, Medium, and Low. Users can add, update, delete, complete, and view tasks sorted by urgency. The system applies Object-Oriented Programming (OOP) principles such as encapsulation and abstraction to ensure clean structure and maintainability. Priority-based scheduling allows important tasks to appear first, improving productivity and task organization.
