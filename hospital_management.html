Hospital Management System:

The Hospital Patient Management System is a scalable database-driven solution designed to efficiently manage hospital operations such as patient records, doctors, appointments, admissions, prescriptions, billing, and medical records. The system is implemented using MySQL Workbench and follows a structured database design approach involving requirement analysis, ER modeling, normalization, and physical schema implementation. To ensure reliability and data safety, the system incorporates transaction management, concurrency control, and backup and recovery mechanisms. In addition, a conceptual NoSQL integration using JSON-based documents is proposed to handle unstructured medical data such as radiology reports and lab results. The hybrid design demonstrates how relational and non-relational models can work together to support modern healthcare data needs while maintaining data integrity, security, and scalability.

Languages & Technologies Used:

  • MySQL
  • MySQL Workbench
  • EER Diagrams, Normalization (up to 3NF)
